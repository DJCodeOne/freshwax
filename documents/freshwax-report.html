Executive Summary

Fresh Wax is a full-stack e-commerce platform purpose-built for the jungle and drum & bass music community. It combines a record label, digital music store, vinyl marketplace, DJ streaming platform, and social community into a single cohesive experience.

The platform was designed and built from November 2025 to March 2026 using modern web technologies, with a focus on performance, security, and accessibility. It runs entirely on Cloudflare's edge network, meaning pages load in milliseconds from servers closest to each visitor worldwide.

In plain English: Fresh Wax is like having Bandcamp + Discogs + Mixcloud + Twitch all rolled into one website, but specifically built for the jungle and drum & bass scene. Artists can sell music, DJs can stream live, collectors can trade vinyl, and fans can discover new tracks — all in one place.

Website Sections

Fresh Wax is organised into distinct sections, each serving a specific purpose within the music ecosystem.

🎵

All Releases (Digital Store)

Browse and buy digital releases grouped by record label. Each release has track previews (60-second clips), artwork, and instant download in MP3/WAV after purchase. Supports "Name Your Own Price" for artists who want flexible pricing. Releases are sorted by label with a quick-jump dropdown.

📀

Vinyl Crates (Marketplace)

A Discogs-style marketplace where collectors and sellers list physical vinyl records. Sellers manage their own store with listings, pricing, and order fulfilment. Buyers browse by condition, label, and price. Each listing has images, condition grading, and shipping costs.

🎧

DJ Mixes

A dedicated section for DJ mixes with streaming playback, artwork, tracklists, and download options. DJs upload their sets with cover art and the system generates 400x400 thumbnails. Mixes can be liked, commented on (with GIF support), and shared to social media.

📡

Live Streaming

Watch live DJ sets with real-time chat, emoji reactions, shoutouts, and viewer counts. Supports HLS video (via OBS), Icecast audio (via BUTT), external relay stations (like Underground Lair), and browser-based streaming (WebRTC/WHIP for mobile DJs). Includes a fullscreen mode with its own chat.

🎛️

DJ Lobby

A private backstage area for DJs. Book streaming slots, see who's up next, monitor audio levels with VU meters, preview your stream before going live, and chat with other DJs. Supports OBS, BUTT, relay, and browser broadcast modes.

🔀

Playlist & Shuffle

A global playlist system that plays 60-second preview clips from the music catalogue when no DJ is streaming. Users can add YouTube videos to the queue, save personal playlists, and the system auto-plays random tracks with artist/title display and countdown timers.

👕

Merchandise Store

Official jungle and drum & bass merchandise — t-shirts, hoodies, hats. Supports multiple sizes, colours, and variants with stock management. Colour swatches with thumbnail sync, brand filtering, and supplier management for dropshipping partners.

📊

Charts & Pre-Orders

Weekly sales charts showing trending releases, and a pre-orders section for upcoming music. Charts update automatically based on sales data. Pre-orders display countdown timers to release date.

🎁

Gift Cards

Digital gift cards that can be purchased and redeemed against any store purchase. Custom amounts, personalised messages, and email delivery. Balance tracked per user account with secure redemption codes.

📝

Blog

News, interviews, and scene updates. Blog posts support rich formatting, images, and are fully SEO-optimised with structured data for search engines.

👤

User Dashboard

Manage your profile, view order history, track downloads, manage social connections (followers/following), view wishlist, and handle subscription (Plus tier for cloud playlist sync and extra features).

⚙️

Admin Dashboard

Full back-office with user management, partner management, inventory control, order processing, sales reports, royalty tracking, error monitoring, streaming management, and newsletter tools.

Security Measures

The platform handles real money, stock inventory, and customer personal data. Security has been a priority from day one, with a full penetration test completed in February 2026.

🔐

Authentication

Firebase Auth with Google Sign-In. All API endpoints verify identity via signed tokens. Admin endpoints require additional admin key verification.

💳

Payment Security

All prices recalculated server-side (never trust the client). Stripe webhook signature verification with timing-safe comparison. PayPal capture amount validation.

🛡️

Data Protection

CSRF double-submit cookies, rate limiting on all endpoints (5-120 req/min depending on sensitivity), parameterised SQL queries, input validation with Zod schemas.

📋

GDPR Compliance

Cookie consent with granular controls, data export, account deletion, double opt-in newsletter, age gate, privacy policy. Users can request full data export or deletion.

Infrastructure & DevOps

🚀

CI/CD Pipeline

4 GitHub Actions workflows: lint → typecheck → test → build → deploy to staging → E2E smoke tests → production deploy with health check and automatic rollback.

💾

Database Strategy

D1 (SQLite at edge) as primary data source for fast reads. Firebase Firestore as fallback. 12 D1 tables with 13 versioned migrations. Weekly automated SQL backups to GitHub.

📊

Error Monitoring

Errors logged to D1 with fingerprinting and deduplication. Webhook alerts when error rate spikes (10+/hour). 30-day automatic retention cleanup. Admin dashboard for error review.

⚡

Caching

Three-tier cache: in-memory (~0ms) → Cloudflare KV (~30ms) → D1/Firebase (~300ms). Releases, mixes, and merch cached for 30 minutes. Empty results never cached to prevent stale data.
